# Currency Converter

A simple currency converter web app using HTML, CSS, and JavaScript.

## Features
-Convert between currencies
-Real-time exchange rates (API)
-Clean UI

## How to run
Open index.html in browser

## Tech Used
-HTML
-CSS
-JavaScript

## About
A currency convertor that converts currency from one currency to another,using api from flagsapi.com (for flags image) and https://cdn.jsdelivr.net/npm/@fawazahmed0/currency-api@latest/v1/currencies/{currency}.json to get the current rate of conversions.
